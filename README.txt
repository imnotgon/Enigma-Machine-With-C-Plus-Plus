Esse projeto se baseia na tentaiva de construção da máquina de criptografia ENIGMA
da segunda guerra mundial, utilizando a linguagem C++. Sintam-se livres para contribuir.

On this project, I will atempt to build the ENIGMA machine from the second world war using
C++. Feel free to help and contribute.
